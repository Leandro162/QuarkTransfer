# 分享访问追踪

打开追踪面板：

```text
http://127.0.0.1:8765/tracker
```

在页面里录入夸克网盘原链接后，工具会生成一个本地追踪链接，例如：

```text
http://127.0.0.1:8765/t/xxxxxxxx
```

对外分享这个追踪链接。别人访问时，服务会先记录访问次数，再自动跳转到原始夸克链接。

页面顶部会显示：

- 所有链接今日访问次数
- 所有链接累计访问次数
- 当前追踪的资源数量

下方表格会显示每个资源的今日访问和累计访问。统计数据保存在本地：

```text
config\tracked_links.json
```

注意：只有通过追踪链接访问才会计数；如果直接分享原始夸克链接，本工具无法从夸克官方侧读取访问次数。
